# 广州市中考录取数据库设计说明

**数据来源**：广州市招生考试委员会办公室（gzzk.gz.gov.cn）  
**覆盖年份**：2021、2022、2023、2024、2025  
**覆盖批次**：第一批次 ～ 第四批次  
**数据库文件**：`guangzhou_admission.db`（SQLite）

---

## 一、各批次数据特点分析

| 批次 | 招生方式 | 核心特点 | 数据结构决策 |
|------|----------|----------|------------|
| **第一批次** | 特长生、自主招生、外语艺术类、港澳子弟班、中职三二分段/中本贯通 | 多种招生类型并列，计分方式各异（合成百分制 / 双分制 / 统一分） | 拆分为5张子表 |
| **第二批次** | 名额分配（指标到校） | 每所高中在每所初中单独设线，涉及"送生初中×招生高中"笛卡尔积，数据量巨大，官方以PDF发布 | 独立表留结构占位，字段含初中名 |
| **第三批次** | 统一招生（普通高中） | 公办高中按**户籍生/非户籍生/外区生**三类分列；民办/中外合作无分类，仅一套分数 | 公办民办分2张表 |
| **第四批次** | 普通高中/综合高中 | 招生范围为所在区（非全市），综合高中含**专业名称**，无外区生字段 | 独立的公办/民办2张表；is_comprehensive标记综合高中 |

## 二、数据库表结构

### 2.1 公共维度表

#### `schools` — 学校主表

| 字段 | 类型 | 说明 |
|------|------|------|
| school_id | INTEGER PK | 自增主键 |
| school_name | TEXT | 含校区全名（唯一键），如"华南师范大学附属中学（石牌校区）" |
| base_name | TEXT | 去校区后的基础名称 |
| campus_name | TEXT | 校区名称，NULL表示无校区 |
| school_type | TEXT | 公办 / 民办 / 中外合作 |
| district | TEXT | 所属区 |
| is_vocational | INTEGER | 0=普通高中，1=职校 |

> 共收录学校 **192所**

#### `control_lines` — 录取控制分数线

| 字段 | 说明 |
|------|------|
| year | 年份 |
| line_type | 线型（如"普高最低控制线"、"普高第一梯度"） |
| score | 分数 |

广州普高录取控制线历年梯度（户籍生）：

| 梯度 | 2021 | 2022 | 2023 | 2024 | 2025 |
|------|------|------|------|------|------|
| 最低控制线 | 515 | 545 | 465 | 502 | 487 |
| 第一梯度 | 690 | 710 | ~695 | 707 | 707 |
| 第二梯度 | 650 | 670 | 675 | 662 | 667 |
| 第三梯度 | 610 | 630 | 635 | 622 | 627 |
| 第四梯度 | 570 | 590 | 595 | 582 | 587 |

### 2.2 第一批次（5张子表）

#### `batch1_special` — 特长生

**计分规则**：合成成绩（满分100分）
- 体育类：中考×40% + 专业测试×60%
- 艺术类：中考×50% + 专业测试×50%

| 核心字段 | 说明 |
|----------|------|
| project_type | 田径/游泳/合唱/美术等 |
| category | 体育 / 艺术 |
| composite_score | 合成成绩（REAL，四位小数） |

#### `batch1_autonomous` — 自主招生

**计分规则**：合成成绩 = 中考÷810×100×70% + 综合能力考核×30%

| 核心字段 | 说明 |
|----------|------|
| student_type | 户籍生 / 非户籍生 |
| exam_score | 末位考生中考分（810分制） |
| ability_score | 末位考生综合能力考核分（100分制） |
| composite_score | 末位合成成绩 |

#### `batch1_arts` — 外语艺术类

| 核心字段 | 说明 |
|----------|------|
| art_type | 外语/美术类/传媒/艺术 |
| min_english | 英语单科最低要求 |
| min_professional | 专业测试最低要求 |

#### `batch1_hkmo` — 港澳子弟班

招生对象：港澳居民子女，单独投档，只有总分线。

#### `batch1_vocational` — 中职三二分段/中本贯通

| 核心字段 | 说明 |
|----------|------|
| major_name | 专业名称 |
| program_type | 三二分段/省重专业/中本贯通/中高企长学制 |
| target_college | 中本贯通对口本科院校 |

---

### 2.3 第二批次 — 名额分配

#### `batch2_quota`

| 核心字段 | 说明 |
|----------|------|
| senior_school_id | 高中学校ID（外键） |
| junior_school_name | 初中学校名（送生校，存文本） |
| student_type | `不限`（第二批次名额分配不区分户籍） |
| min_score | 最低录取分数 |
| min_score_same_rank | 最低分数同分序号 |
| last_score | 末位考生分数 |
| last_wish_rank | 末位考生志愿序号 |
| last_score_same_rank | 末位考生分数同分序号 |
| data_source | `PDF官方附件` |

> **✅ 已完整导入**：2023/2024/2025年官方PDF全量数据，共 **17,395条**记录，覆盖所有招生高中×送生初中组合。  
> 
> **PDF来源**（按招生学校排序版本）：
> - 2023年：`https://gzzk.gz.gov.cn/attachment/7/7391/7391064/9106619.pdf`
> - 2024年：`https://gzzk.gz.gov.cn/attachment/7/7637/7637799/9768847.pdf`
> - 2025年：`https://gzzk.gz.gov.cn/attachment/7/7848/7848251/10363933.pdf`
>
> **解析工具**：`parse_batch2_pdf.py`（pdfplumber）

### 2.4 第三批次（2张表）

#### `batch3_public` — 统一招生·公办

**✅ 已完整导入全量数据（1095条）**：2021年156条 / 2022年167条 / 2023年244条 / 2024年253条 / 2025年275条，严格对应官方网页。

**核心特点**：三类学生分列

| student_type | 定义 | 覆盖年份 |
|--------------|------|---------|
| 户籍生 | 广州市户籍（含政策照顾学生） | 2021/2022/2023/2024/2025 |
| 非户籍生 | 符合条件的随迁子女 | 2021/2022/2023/2024/2025 |
| 外区生 | 区属示范高中录取的外区生 | 2023/2024/2025 |

**学校覆盖**：
- 2021年：约78所
- 2022年：约84所
- 2023年：89所（含66条外区生记录）
- 2024年：91所（含71条外区生记录）
- 2025年：99所（官方命名）→ 数据库99条

**数据来源**：
- 2021-2022年：官方网页图片（PNG格式）通过 OCR.space API 识别 + 多源新闻交叉验证
- 2023-2025年：官方网页 HTML 表格直接解析

**命名说明**：
- 2023年广雅"本部校区"即2024年起的"荔湾校区"，数据库按各年官方命名存储
- 省市属高中（华附/省实/广雅/执信/二中/广大附等）无外区生，不写入外区生记录
- `--`（无考生录取）的记录不写入数据库

每条记录含分数字段：
- `min_score`：最低录取分数
- `last_score`：末位考生分数（官方未公布同分序号，与min_score相同时已合并）

#### `batch3_private` — 统一招生·民办/中外合作

| sub_type示例 | 说明 |
|-------------|------|
| 公费班 | 政府补贴计划 |
| AP课程/A-Level/IB-DP | 国际课程合作班 |
| 普通高中 | 标准民办计划 |

---

### 2.5 第四批次（2张表）

#### `batch4_public` — 普通/综合高中·公办

与第三批次公办结构相似，新增字段：
- `is_comprehensive`：0=普通高中，1=综合高中
- `major_name`：综合高中专业名称（普通高中为NULL）
- 无`外区生`字段（第四批次为区属学校，仅本区招生）

#### `batch4_private` — 普通高中·民办/中外合作

结构同第三批次民办，独立成表以便按批次独立分析。

---

### 2.6 招生计划（报考指南）

#### `enrollment_plan` — 各校各批次招生计划数

**✅ 已完整导入（851条）**：2021-2025年第三/四批次招生计划，来自官方《报考指南》PDF。

| 字段 | 类型 | 说明 |
|------|------|------|
| year | INTEGER | 年份 |
| batch | TEXT | 第三批次 / 第四批次 |
| school_name | TEXT | 学校全称（含校区） |
| school_type | TEXT | 公办 / 民办 / 中外合作 |
| school_level | TEXT | 国家级示范性普通高中 / 广州市示范性普通高中 / 其他普通高中 等 |
| district | TEXT | 所在区 |
| plan_total | INTEGER | 招生计划总数（户籍生计划） |
| plan_suqian | INTEGER | 随迁子女最大招生计划（民办为NULL） |
| plan_waiqu | INTEGER | 外区招生最大计划数（第四批次为0；省市属"不限"为NULL） |
| scope_huji | TEXT | 户籍生招生范围 |
| scope_suqian | TEXT | 随迁子女招生范围 |

**数据量**：
- 第三批次公办：2021年78条 / 2022年83条 / 2023年86条 / 2024年91条 / 2025年99条
- 第三批次民办/中外合作：逐年递增（9→10→21→39→53条）
- 第四批次公办：2021年26条 / 2022年24条 / 2023年22条 / 2024年33条 / 2025年34条
- 第四批次民办/中外合作：逐年递增（19→21→31→32→39条）

**数据来源**：广州市招生考试委员会办公室官方PDF《报考指南》（2021-2025年），pdfplumber解析。

**注意事项**：
- `plan_waiqu = NULL` 表示"不限"（省市属高中面向全市招生，无外区限制）
- 民办学校 `plan_suqian = NULL`（民办不区分户籍/随迁子女计划）
- 中外合作项目从2022年开始在第三批次出现
- 综合高中（2023年起）的 `school_type` 为"公办"，学校名含"综合高中"或"（XX专业）"

## 五、常用查询示例

```sql
-- 1. 查询2025年第三批次户籍生最高分学校
SELECT school_name, min_score
FROM v_batch3_public_huji
WHERE year = 2025
LIMIT 20;

-- 2. 查询某校五年分数趋势
SELECT school_name, score_2021, score_2022, score_2023, score_2024, score_2025
FROM v_score_trend
WHERE school_name LIKE '%执信%';

-- 3. 查询第四批次公办综合高中2025年各专业分数
SELECT s.school_name, b.major_name, b.student_type, b.min_score
FROM batch4_public b
JOIN schools s ON b.school_id = s.school_id
WHERE b.year = 2025 AND b.is_comprehensive = 1
ORDER BY b.min_score DESC;

-- 4. 查询2025年港澳子弟班录取门槛
SELECT s.school_name, h.min_score, h.last_score
FROM batch1_hkmo h
JOIN schools s ON h.school_id = s.school_id
WHERE h.year = 2025;

-- 5. 第四批次非户籍生与户籍生分差（2025）
SELECT s.school_name,
    MAX(CASE WHEN b.student_type='户籍生' THEN b.min_score END) as huji,
    MAX(CASE WHEN b.student_type='非户籍生' THEN b.min_score END) as feihuji,
    MAX(CASE WHEN b.student_type='非户籍生' THEN b.min_score END) -
    MAX(CASE WHEN b.student_type='户籍生' THEN b.min_score END) as gap
FROM batch4_public b
JOIN schools s ON b.school_id = s.school_id
WHERE b.year = 2025 AND b.is_comprehensive = 0
GROUP BY b.school_id
ORDER BY gap DESC
LIMIT 10;

-- 6. 查询某初中2025年送往各高中的最低分（第二批次名额分配）
SELECT s.school_name as 招生高中, b.min_score as 最低分, b.last_score as 末位分
FROM batch2_quota b
JOIN schools s ON b.senior_school_id = s.school_id
WHERE b.year = 2025 AND b.junior_school_name LIKE '%执信%'
ORDER BY b.min_score DESC;

-- 7. 第二批次：某高中历年在同一初中的录取分趋势
SELECT b.year, b.min_score, b.last_score
FROM batch2_quota b
JOIN schools s ON b.senior_school_id = s.school_id
WHERE s.school_name LIKE '%广州市第二中学%'
  AND b.junior_school_name = '广州市第七中学'
ORDER BY b.year;

-- 8. 第二批次最难进的初中（被顶尖高中录取分最高）
SELECT b.junior_school_name, s.school_name as 招生高中, b.year, b.min_score
FROM batch2_quota b
JOIN schools s ON b.senior_school_id = s.school_id
WHERE b.year = 2025 AND b.min_score >= 770
ORDER BY b.min_score DESC
LIMIT 20;
```
